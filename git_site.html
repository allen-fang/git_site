What is Git?

Git is a version contorl system; you can thinkn of it as a series of snapshots (comits) of your code base. With the creation of each asnapshot you build out a whole applicaiton, seeing the growth of a full app. During this time you can make branches to experienemtn and come back to a previous snapshots you took or merge those experiements into your app.

What is Github?

GitHub is a code hosting platform for version control and collaboration. Itlets you and others work toegheter on projects from anywhere using those code snapshots that you took during development.
